Project Summary

The topic of my site is fashion trends in the Fall/Winter 2023 season, which include the keywords of this season’s fashion, popular styles of outfits, and colors and material of outfits that are commonly used this season. The purpose of my site is to allow the viewers to gain a better understanding of which designs and styles of clothing are currently favored, which could also help them choose which outfits to wear if they would like to follow these trends. In my main page, I included a general description of the changes in fashion trends this season, which serve as an introduction to the content of my website. The following pages discuss more detailed information about which styles, colors, and materials are popular this season.

The extras I used in my site are transitions & transformations and jQuery. In the keyword section of the main page, I used transitions & transformations to translate each image when the cursor hovers over it. JQuery was also used to make each image clickable, and the description of each image that is clicked would be displayed. On the “Styles” page, I also included transitions & transformations to enlarge each image when hovered over, allowing viewers to see it more closely. Lastly, on the “Colors/Material” page, I used jQuery to make each color/material clickable so that the name of the color/material that is clicked would be displayed.
